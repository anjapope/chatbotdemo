How to use this demo

This site demonstrates a small workflow for collecting peer critiques and suggestions for model responses.

Chat

  1. Open Chat and ask the assistant a question.
  2. After the assistant replies, use the rating UI under the assistant message to rate factuality, clarity, and ethics.
  3. Optionally suggest a revision by clicking "Suggest an edit" on the assistant message.

Peer Review

  1. Open Peer Review to see a list of example prompts and multiple candidate responses.
  2. Click an item to view responses. Use Select to indicate which response you are critiquing.
  3. Write a critique in the box and click Submit critique.
